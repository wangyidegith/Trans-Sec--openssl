零 数学概念
（零）数学原理：椭圆曲线数学。


（一）生成公私钥对（接收方）
0 选择一个椭圆曲线，并选择相关参数
（0）椭圆曲线：y^2 = x^3 + ax + b；
（1）设置生成点G：该点是椭圆曲线上的一个特定点(x, y)，x、y的长度 == 密钥长度；
（2）设置阶n：生成点G的阶，即nG = 无穷大，因而n是一个大素数；
注意：G和n是openssl ecc、sm2等算法规定的，不由用户设置。

1 生成私钥d：1 <= d < n

2 生成标准公钥Q = d 点乘 G

注意：以上总共四个参数G、n、d、Q，除了d是私钥不能公开，剩下的都公开。


（二）如何使用公私钥对加解密？
0 如何使用公钥进行加密？（发送方）
（0）假设明文为plaintext；
（1）生成一个随机数k：1 <= k < n；
（2）传输数R = k 点乘 G；
（3）计算共享密钥：S = k 点乘 Q；
（4）加密数据：cpiphertext = AES_encrypt(plaintext, S)；
（5）将[ciphertext, R]发往接收方。

1 如何使用私钥进行解密？（接收方）
（0）拿到[ciphertext, R]；
（1）计算共享密钥：S = d 点乘 R；
（2）解密数据：plaintext = AES_decrypt(ciphertext, S)；
